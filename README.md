Apresentação do sistema
Nome: Organiza Fácil
Problema resolvido: A distribuição das tarefas muitas vezes salvo em vários aplicativos diferentes.
Público-alvo: Todas as pessoas entre 13 e 80 anos, que queiram organizar sua rotina pessoal de
forma online.
Objetivo do sistema: Ajudar pessoas a organizar suas tarefas diárias em um só lugar, de forma
online.

Ferramentas usadas
Será usado HTML, CS e JS

Requisitos Funcionais 
RF_01: Cadastrar tarefas: O sistema deve permitir o cadastro de uma nova tarefa, com os campos
“Nome” e “Descrição”

RF_02: Listar tarefas: O sistema deve permitir a listagem das tarefas por ordem alfabética.

RF_03: Marcar como concluídas: O sistema deve permitir ao usuário marcar uma tarefa como
concluída

RF_04: Excluir tarefas: O sistema deve permitir ao usuário a exclusão de uma tarefa. 

O sistema não conta com tela e login, notificação de confirmação de "Tarefa Excluída" ou "Tarefa Cadastrada", e ainda sem opções de organização dentro do site como pastas "Trabalho", "Escola", etc. O Front ainda é vago, 
com estrutura visual simples, porém objetiva.
